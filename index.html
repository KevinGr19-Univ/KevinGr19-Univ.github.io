Personal project

Le Démineur

"Le Démineur" (The Minesweeper) is a turn-based multiplayer minesweeper, where the goal is to clear the grid without touching bombs, to be the last one standing.


You would not want to be the one losing though, the losers being roughly punished...


Created with the C# language, along with the Unity Engine.

University project

HypoluxAdventure

HypoluxAdventure is a rogue-like game where Hypolux (you), ventures in a deep cave, looking for a treasure.

Explore random mazes, find chests and get better equipment, because nearby monsters will not rest.


Created with Noa GUILLOT and Mathieu ROSTAING, using the C# language and the Monogame framework.

Download the game (GitHub repository)
